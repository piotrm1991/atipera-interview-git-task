# GitHub Api Project - Atipera interview task

## 1. Environment:

```
Language:   Java 25
```

## 2. Build application

Run Application with Intellij IDEA functionality.

Project should start as expected.

## 3. Description

The "GitHub Api Project" application provides the ability to get the list of repositories of given user.

## 4. API Endpoint

**4.1** The API endpoint for viewing user repositories: `/api/repos/{username}`

**4.2** HTTP Methods

- `GET`: Retrieve list of user repositories.
